#  Gilded Chess Salon

> A visually immersive 3D chess experience designed to transform a familiar game into a refined, atmospheric, and engaging digital experience.

🔗 **Live Demo:** [Play Gilded Chess Salon](https://royal-chess-glow.lovable.app/)

---

## Overview

Gilded Chess Salon is an interactive 3D chess experience focused on creating a visually immersive and refined gameplay environment.

The project explores how thoughtful UI/UX design, visual hierarchy, contrast, and interactive elements can transform a familiar digital experience into something more atmospheric and engaging.

---

## Features

*  Interactive chess gameplay
* Immersive chess pieces
* Luxury-inspired visual direction
*  High-contrast chessboard design
*  Atmospheric and engaging user experience
* Responsive interface
* Modern interactive frontend experience
